## Local setup

### Requirements
- node >=20.x
- npm >=10.x

### Running development environment ###

- clone this repository
- run `npm install`
- copy `.env.sample` file to `.env`
- in `.env` add required data
- run `npm install -g @sanity/cli` to install cli
- run `sanity login` then select email\password for authentication
- run `npm run dev` command to start development environment
